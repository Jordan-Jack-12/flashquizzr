# FlashQuizzr

A Flash Card and Quiz generator. It use AI to generator quizzes and flash card for revision and active recall. It exports the flash card for popular spaced repetition software. It have its own spaced repetition feature to reduce hassle of using different platforms.

## Contributing

1. Fork this repository.

2. Go to the forked repository and clone the repository.

`git clone https://github.com/<your_username>/flashquizzr.git`

3. Install the dependencies.

```
cd project_directory

npm install
```

4. Run the project in dev mode.

`npm run dev`

5. Make some changes or fix issues.

6. Commit the changes.

7. Pull request the commits.

*this is development git repository for the web app*
